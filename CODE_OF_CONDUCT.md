# Code of Conduct

This project follows the
[GNOME Code of Conduct](https://conduct.gnome.org/).

In short: be respectful, be kind, assume good faith, and help make this a
welcoming community for everyone. Harassment and exclusionary behavior are not
tolerated.

Report concerns to the maintainers at **ned.tabulov@gmail.com**.
